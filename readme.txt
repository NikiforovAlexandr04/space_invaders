Описание:
Игра Space Invaders, в которой управляя космическим кораблем необходимо убивать монстров и зарабатывать очки.

Основным файлом при запуске которого запускается игра является - logic_game.py

Состав:
logic_game.py - основной файл, запускающий игру 
interface.py - файл, отрисовывающий графический интерфейс игры
player_shot.py - класс выстрела игрока
music_logic.py - класс управления музыкой
tick.py - класс отвечающий за обновление игровых компанентов
save_logic - класс описывающий сохранение
pictures - папка с картинками 
bunkers.py - файл с логическим расположением бункеров
monsters.py - файл с логикой реализации монстров
monster_shell.py - файл с логическим содержанием снарядов монстров
NLO.py - файл с логикой реализации НЛО
player.py - файл с логикой реализации игрока
shell.py - файл с логикой реализации снарядов игрока
tests.py - файл с тестами

Управление:
"A" - движение влево
"D" - движение вправо
"Space" - выстрел
"P" - пауза
"R" - перезапуск
"T" - таблица рекордов
"S" - сохранение игры
"F" - возобновление игры
